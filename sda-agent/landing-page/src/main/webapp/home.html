IU SEAD Cloud

About

IU SEAD Cloud provides a default publication solution for SEAD-affiliated users. It relies on the Indiana University Scholarly Data Archive (SDA) infrastructure, a distributed storage service co-located at IU data centers in Bloomington and Indianapolis, arranged in large files, with two copies of data made by default.
